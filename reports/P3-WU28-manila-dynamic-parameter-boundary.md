# P3-WU28 Manila dynamic-parameter boundary

Status: useful external sensitivity values found; Manila-specific production
rise time and rupture velocity remain unresolved.

Ren et al. (2019) directly studies how kinematic rupture affects Manila-trench
tsunami propagation, but the accessible primary record does not expose a
reproducible Manila event-specific velocity/rise-time table. Its 3.2--32 s rise-
time range refers to the 2011 Tohoku source and is not promoted to Manila.

Ye et al. (2016) tests 2.0, 2.5 and 3.0 km/s rupture velocities in general large-
megathrust finite-fault inversions. Those values can be used only as explicitly
labelled external sensitivity branches. They are not a Manila prior, posterior,
or default scenario parameter.

Thus literature work can narrow and document the sensitivity experiment now,
but it cannot close the production PEGS scenario gate. That gate still needs an
auditable Manila finite-fault dynamic source or a pre-registered scaling law
with uncertainty.
