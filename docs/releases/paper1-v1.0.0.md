# Paper 1 v1.0.0 release notes

This release freezes the reproducible technical-preprint package for
“Frequency-Coverage Limits in Assessing Gravity Signals from Oceanic Mass
Redistribution.”

## Included

- 1,446-record direct-radial-gravity representative scenario ensemble;
- P1-E008 process-specific 50/75/90/95% frequency requirements;
- P1-E009 Helgoland component audit;
- P1-E010 primary-source instrument-band correction and spectral-convention
  robustness audit;
- five main figures, two supplementary figures and compiled PDF;
- structured 46-entry literature matrix and observable ledger;
- disclosed AI scientific review record;
- one-command figure/audit/PDF reproduction workflow;
- MIT code licence, CC BY 4.0 original-document declaration and citation
  metadata.

## Scientific boundary

The release is a frequency-coverage framework, not a decision-grade
detectability atlas. It reports direct radial gravity for all six families.
Elastic gravity and displacement are separately validated for the Helgoland
case and are not inferred for every process. No result is a universal
detectable/undetectable classification.

## External boundary

The release is citable by Git tag. Zenodo metadata are prepared, but no DOI is
claimed until a repository owner enables Zenodo/GitHub integration and performs
the authenticated deposition. Named authorship and affiliations remain a
journal-submission signature step.
